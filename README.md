Pages incluses offre un moyen simple de servir des pages HTML ou des scripts PHP sur des pages publiques du blog. Il permet :

* d'inclure un fichier PHP dans une page. Celui-ci sera exécuté côté public. Cela peut servir par exemple à créer un formulaire de contact.
* ou bien d'inclure un fichier HTML ordinaire, ou un fichier HTML comprenant des balises de template de Dotclear dans une page. Ce fichier sera alors rendu côté public.


### *Included pages*

*Included aims to provide a simple way to serve HTML templates and PHP scripts. It allows to:*

* *include a PHP file in a page, it will be executed on public side. It may be used for example to create a contact form.*
* *or include any HTML file, or a HTML file with Dotclear's template tags, which will be executed on public page as well.*
