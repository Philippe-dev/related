Pages incluses offre un moyen simple de servir des pages HTML ou des scripts PHP sur des pages publiques du blog. Il permet :

* d'inclure un fichier *.php dans une page. Celui-ci sera exécuté côté public. Cela peut servir par exemple à créer un formulaire de contact.
* d'inclure un fichier *.html ordinaire, ou un fichier *.html comprenant des balises de template dans une page. Ce fichier sera alors rendu côté public.

# Included pages

Included aims to provide a simple way to serve HTML templates and PHP scripts.

* It is possible to include a *.php file in a page, it will be executed on public side. It may be used for example to create a contact form.
* It is also possible to include any *.html ordinary file, or a *.html file with Dotclear's template tags, which will be executed on public page as well.
