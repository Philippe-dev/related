Présentation

Les pages connexes
Ce plugin permet de créer et éditer des pages connexes. Les pages connexes peuvent être créées à partir d'un billet, ou par l'inclusion d'un fichier PHP ou HTML.
Page d'après un billet
La page est éditable comme un billet ordinaire.
Page incluse
Il est possible d'inclure un fichier *.php dans la page connexe, qui sera interprétée côté public. Cela peut servir par exemple à créer un formulaire de contact. Il est aussi possible d'inclure un fichier *.html ordinaire, ou un fichier *.html comportant des balises de template de Dotclear, qui seront alors interprétées côté public.

Paramètres

Activation
Le plugin doit être activé pour fonctionner sur le blog. La désactivation ne supprime pas les pages connexes mais empêche seulement leur affichage.
Chemin du dépôt des pages incluses
Permet de choisir le répertoire où seront stockées les pages connexes.
Préfixe d'URL
Permet de déterminer la chaîne de caractères spécifique dans les urls des pages connexes (par défaut : static).

Gérer les pages

Actions par lot sur les éléments
Il est possible d'effectuer un ensemble d'actions sur plusieurs pages connexes, d'un seul coup. Les actions possibles, comme pour les billets, dépendent des permissions de l'utilisateur.
Créer une nouvelle page connexe
Choisir l'édition directe pour une nouvelle page d'après un billet, ou la création d'une page incluse par l'ajout d'un fichier *.php ou *.html.

Ordonnancer la liste publique

Tri
Permet de déterminer l'ordre des pages dans la liste affichée côté public (widget).
Visibilité
Permet de déterminer quelles sont les pages affichées dans la liste des pages connexes côté public (widget).

Widget

Ce plugin prévoit un widget que vous pourrez placer dans le bandeau de votre choix depuis le panneau Widget de présentation. La liste des pages listées dépendra des choix que vous aurez faits dans "ordonnancer la liste publique"

Fichier de template

Ce plugin ajoute un fichier de thème, external.html. Ce ficher est situé dans le répertoire default-templates du plugin. Vous pouvez le personnaliser en en en plaçant une copie dans le répertoire du thème que vous utilisez puis en y effectuant vos modifications
