Paramètres

Activation
Le plugin doit être activé pour fonctionner sur le blog. La désactivation ne supprime pas les pages connexes mais empêche seulement leur affichage.
Chemin du dépôt des fichiers inclus
Permet de choisir le répertoire où seront stockés les fichiers.
Préfixe d'URL
Permet de déterminer la chaîne de caractères spécifique dans les urls des pages connexes (par défaut : static).

Fichier de template

Ce plugin ajoute un fichier de template nommé external.html. Ce ficher est situé dans le répertoire default-templates du plugin. Vous pouvez le personnaliser en plaçant une copie du fichier dans le répertoire tpl du thème que vous utilisez puis en y effectuant vos modifications
