Présentation

Les pages incluses
Ce plugin permet de créer et éditer des pages incluses, qui permettent d'afficher un fichier PHP ou HTML côté public.
Fichier inclus
Il est possible d'inclure un fichier *.php dans la page incluse, qui sera interprété côté public. Cela peut servir par exemple à créer un formulaire de contact. Il est aussi possible d'inclure un fichier *.html ordinaire, ou un fichier *.html comportant des balises de template de Dotclear, qui seront alors aussi interprétés côté public.

Gérer les pages

Actions par lot sur les éléments
Il est possible d'effectuer un ensemble d'actions sur plusieurs pages incluses, d'un seul coup. Les actions possibles, comme pour les billets, dépendent des permissions de l'utilisateur.

Ordre des pages

Ordre
Permet de déterminer l'ordre des pages dans la liste affichée côté public (widget). Déplacez les pages, puis enregistrez leur ordre

Widget

Ce plugin prévoit un widget que vous pourrez placer dans le bandeau de votre choix depuis le panneau Widget de présentation.
