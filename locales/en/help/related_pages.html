Use

Included pages
This plugin allows to create and edit included pages. Pages may include *.php or *.html files.
Included file
It is possible to include a *.php file in a included page, it will be executed on public side. It may be used for example to create a contact form. It is also possible to include any *.html ordinary file, or a *.html file with Dotclear's template tags, which will be executed on public page as well.

Manage pages

Selected pages actions
If your user permission level allows it, you can perform batch operations on the pages you select in the list..
Create a new page
Choose creating and editing a page as a post, or by including *.php or *.html file.

Arrange public list

Sort
Allows to change order of included pages on public side (widget).
Visibility
Allows to choose which included pages will be displayed on public side (widget).

Widget

This plugin includes a widget to display a list of included pages on public side. The list will depend on choices made in "Arrange public list"
